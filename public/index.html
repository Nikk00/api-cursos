Documentación

REST

La URL base contiene todos los cursos disponibles. Todas las solicitudes son GET solicitudes y repasan https. Todas las respuestas devolverán datos en formato JSON. 

                GET  https://api-cursos.vercel.app/api/cursos
            
                
                {
                    "title": String,
                    "url": String,
                    "urlTitle": String
                }

CONSIGUE UN SOLO CURSO

Puedes obtener un solo curso agregando urlTitle como parámetro: /cursos/urlTitle 

                GET  https://api-cursos.vercel.app/api/cursos/:urlTitle
            
                
                {
                    "url": String
                }
